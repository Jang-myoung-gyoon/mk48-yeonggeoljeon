# Battle Floating Text Design

## Goal

전투 HUD에서 공격/책략 피해와 도구 회복 수치를 타일 위에 즉시 보여주고, 위로 떠오르며 사라지는 짧은 전투 피드백을 제공한다.

## Chosen Approach

- 로그 문자열을 파싱하지 않는다.
- 전투 화면 상태가 액션 전후 `BattleState` 의 HP 차이를 계산해 구조화된 플로팅 텍스트 엔트리를 만든다.
- 그리드 셀은 해당 위치의 엔트리를 오버레이로 렌더링한다.

## Why

- 엔진 로그 포맷과 UI 피드백을 결합하지 않기 위해서다.
- 이후 크리티컬, 회피, 버프/디버프 같은 전투 피드백을 추가할 때 같은 구조를 재사용할 수 있다.

## Behavior

- 피해는 `-숫자`, 회복은 `+숫자` 형식으로 표시한다.
- 텍스트는 셀 위에 나타난 뒤 위로 이동하며 페이드아웃된다.
- 약 1초 뒤 상태에서 제거된다.
- 리셋/턴 전환 시 남아 있는 플로팅 텍스트와 타이머는 정리한다.

## Testing

- 위젯 테스트로 공격 후 피해 텍스트가 나타났다 사라지는지 확인한다.
- 위젯 테스트로 도구 사용 후 회복 텍스트가 나타났다 사라지는지 확인한다.
- 전체 `flutter analyze` 와 `flutter test` 로 회귀를 확인한다.
